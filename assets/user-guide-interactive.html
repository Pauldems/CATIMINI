🎯 Comment utiliser ce guide

Cliquez sur les exemples ci-dessous pour voir comment les différents états apparaissent dans le calendrier. Cela vous aidera à comprendre rapidement les codes couleur de l'application.

Comprendre les codes couleur

15
Jour disponible
Vous n'avez aucune indisponibilité ni événement ce jour
15
Indisponibilité
Fond bleu foncé = vous êtes indisponible ce jour
15
Événement planifié
Cercle jaune + point jaune = un événement où vous participez
15
Indisponibilité + Événement
Fond bleu + cercle jaune = vous êtes indisponible ET avez un événement
15
Date sélectionnée
Fond jaune = date actuellement sélectionnée

💡 Bon à savoir

  • Les indisponibilités s'appliquent à tous vos groupes
  • Vous pouvez avoir plusieurs événements le même jour
  • Les événements affichés sont ceux de votre groupe actuel
